# ЭнергоАтлант

Веб-платформа управления строительными проектами для электромонтажной компании.

## Как добавить фото

Услуги: положи файл в `public/img/services/` с именем из списка ниже

| Файл | Услуга |
|---|---|
| `vl.jpg` | Воздушные линии |
| `kl.jpg` | Кабельные линии |
| `tp.jpg` | Трансформаторные подстанции |
| `pnr.jpg` | Электроснабжение объектов |
| `docs.jpg` | Исполнительная документация |
| `project.jpg` | Проектирование |
| `gnb.jpg` | ГНБ (резерв) |

Портфолио: положи файл в `public/img/portfolio/` по маске `obj_{номер}_{фото}.jpg`

Пример для объекта №1: `obj_1_1.jpg`, `obj_1_2.jpg`, `obj_1_3.jpg` ...

Форматы: jpg, webp (рекомендуется webp для скорости)

Размер: не более 500KB на фото, рекомендуемое разрешение 1200x800px
